# gen-search

Generate genealogy search links for a ton of websites.

````
var fsLink = search('familysearch', {
  givenName: "Joe William",
  familyName: "Clark"
});
````

## Schema

* givenName
* familyName
* birthPlace
* birthDate
* deathPlace
* deathDate
* fatherGivenName
* fatherFamilyName
* motherGivenName
* motherFamilyName
* spouseGivenName
* spouseFamilyName
