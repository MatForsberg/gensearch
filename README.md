# gen-search

Generate genealogy search links for a ton of websites.

````
var fsLink = gensearch('familysearch', {
  givenName: "Joe William",
  familyName: "Clark"
});
````

### __[Try the Demo](http://genealogysystems.github.io/gen-search/)__
